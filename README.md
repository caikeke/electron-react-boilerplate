a react electron boilerplate.

using [electron-forge](https://github.com/electron-userland/electron-forge)

includes:

* react && react-dom
* redux && redux-thunk
* react-router-dom
* react-hot-loader
* axios
* eslint
* husky

## install

```
npm install
```

or

```
yarn
```

## develop

```
npm start
```

## build

```
npm run package
```
